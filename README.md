# MangaShelf

A desktop manga browser, downloader, and reader for Windows, powered by the official
[MangaDex API](https://api.mangadex.org/docs/). This is the full rewrite of the old
Manganelo scraper script (which now lives in `legacy/`).

## Features

- **Discover** — front page with Most Popular, Top Rated, and Recently Updated rows
- **Browse & search** — title search plus genre/tag, status, and sort filters
- **Manga pages** — cover, description, rating & follow stats, related series, and
  "more like this" recommendations
- **Downloads** — queue whole series or single chapters; pages are saved as plain
  images (`Manga Library/<Series>/Chapter <n>/001.jpg`) with a live progress queue
- **Built-in reader** — vertical scroll or paged mode, fit/width controls, keyboard
  navigation (←/→, Esc), remembers your spot per series; can also stream chapters
  online without downloading
- **Library shelves** — bookmark any series with a status (Reading / Plan to Read /
  Completed / On Hold / Dropped) via the ☆ button; no download required. Starting
  a "Plan to Read" series auto-moves it to "Reading"
- **Continue Reading** — your exact chapter *and page* are saved for every series
  you open (downloaded or streamed); the Home page has a Continue Reading row that
  drops you right back in
- **Updates feed** — followed series are checked for new chapters automatically on
  launch and every 2 hours (plus on demand); new releases appear in the Updates
  tab with a badge
- **Desktop notifications** — get a Windows notification when followed series drop
  new chapters; click it to jump to the Updates tab. Subscribe/mute per series
  with the 🔔 button, or toggle globally in Settings
- **Downloads tab in Library** — everything you've downloaded, with progress bars
- **Export** — any downloaded chapter (or a whole series) as `.cbz` or `.pdf`
- **Settings** — library folder, content-rating filter, image quality (original /
  data saver), chapter language

> Note: some licensed titles on MangaDex (e.g. big Shonen Jump series) only list
> *external* chapters that point to the publisher's own site. Those can't be
> downloaded; the app marks them "external only".

## Development

```
npm install
npm start        # run the app
npm run dev      # run with devtools + verbose logging
```

Main process code is in `electron/` (API client, download queue, library db,
CBZ/PDF export), the UI in `renderer/` (vanilla JS modules, no build step).

## Building the .exe

```
npm run dist
```

Outputs to `dist/`: a portable `MangaShelf-<version>-portable.exe` (single file,
no install) and an NSIS installer. The icon is generated by
`node scripts/make-icon.js`.

## Storage

- Downloads: `Documents\Manga Library` by default (change in Settings)
- Library index & settings: `%APPDATA%\mangashelf`

## Disclaimer

For personal offline reading. Data and images come from MangaDex — please support
them and the scanlation groups at [mangadex.org](https://mangadex.org). Ensure you
have the necessary permissions to download content in your jurisdiction.
